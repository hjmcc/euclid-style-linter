🔭 Euclid Style Linter

Check LaTeX source against the ECEB Style Guide V5

Runs entirely in your browser. The linter executes locally via Pyodide (Python compiled to WebAssembly): your text is never uploaded and never leaves this page. The only network requests are the one-off downloads of the Python runtime (from the jsDelivr CDN) and the linter script when the page loads. You can verify this in the browser's network inspector. This tool is not an official product of the ECEB.
